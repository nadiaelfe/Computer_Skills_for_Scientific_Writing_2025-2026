# Выполнение лабораторной работы

## 1. Подключение пакетов для работы с таблицами

Для работы с таблицами в LaTeX необходимо подключить пакет array в преамбуле документа. Для профессионального оформления таблиц также используется пакет booktabs.

## 2. Типы выравнивания колонок

В LaTeX существуют три основных типа выравнивания колонок:

$$ \text{Типы колонок} = \{l,\ c,\ r\} $$

где:
- $l$ — выравнивание влево (left)
- $c$ — выравнивание по центру (center)
- $r$ — выравнивание вправо (right)

### 2.1 Выравнивание влево (l)

При использовании левого выравнивания текст в колонках прижимается к левому краю:

| Имя | Фамилия | Возраст |
|:----|:--------|--------:|
| Иван | Петров | 25 |
| Мария | Иванова | 30 |
| Петр | Сидоров | 28 |

### 2.2 Выравнивание по центру (c)

При использовании центрирования текст располагается по центру каждой колонки:

| Имя | Фамилия | Возраст |
|:---:|:-------:|:-------:|
| Иван | Петров | 25 |
| Мария | Иванова | 30 |
| Петр | Сидоров | 28 |

### 2.3 Выравнивание вправо (r)

При использовании правого выравнивания текст прижимается к правому краю колонок:

| Имя | Фамилия | Возраст |
|----:|--------:|--------:|
| Иван | Петров | 25 |
| Мария | Иванова | 30 |
| Петр | Сидоров | 28 |

## 3. Профессиональные линии в таблицах

Для создания профессиональных горизонтальных линий используются специальные команды из пакета booktabs:

$$ \text{Типы линий} = \{\text{\toprule}, \text{\midrule}, \text{\bottomrule}\} $$

Пример таблицы с профессиональными линиями:

| Страна | Столица | Население (млн) |
|:-------|:--------|----------------:|
| Россия | Москва | 146 |
| Франция | Париж | 67 |
| Япония | Токио | 125 |

## 4. Эксперименты с количеством элементов в строках

### 4.1 Недостаточное количество элементов

При создании таблицы с тремя колонками, если в строке указано меньше элементов:

$$ \text{Колонок определено} = 3 $$

$$ \text{Элементов в строке} = 2 $$

Пример некорректной таблицы:

| Имя | Возраст | Город |
|:----|:-------:|:------|
| Иван | 25 | |
| Мария | 30 | Москва |

**Результат:** LaTeX выдает ошибку, но PDF-файл создается. Строка с недостаточными элементами завершается преждевременно, что приводит к смещению последующих данных в неправильные колонки.

### 4.2 Избыточное количество элементов

При указании большего количества элементов, чем определено колонок:

$$ \text{Колонок определено} = 3 $$

$$ \text{Элементов в строке} = 5 $$

$$ \text{Дополнительные элементы} = 2 $$

Пример некорректной таблицы:

| Имя | Возраст | Город |
|:----|:-------:|:------|
| Иван | 25 | Москва | доп1 | доп2 |
| Мария | 30 | СПб | |

**Результат:** LaTeX генерирует ошибку, но компиляция продолжается. Дополнительные элементы выводятся за пределами структуры таблицы.

## 5. Использование команды \multicolumn

Для объединения нескольких колонок в одну используется команда \multicolumn:

$$ \text{\multicolumn}\{n\}\{\text{align}\}\{\text{text}\} $$

где:
- $n$ — количество объединяемых колонок
- $\text{align}$ — выравнивание $(l, c, r)$
- $\text{text}$ — отображаемый текст

### 5.1 Объединение двух колонок

| \multicolumn{2}{|c|}{Личная информация} | Город |
|:---|:---:|:---:|
| Имя | Возраст | |
| Иван | 25 | Москва |
| Мария | 30 | СПб |

Параметры:
$$n = 2,\ \text{align} = c,\ \text{text} = \text{"Личная информация"}$$

### 5.2 Объединение всех трех колонок

| \multicolumn{3}{|c|}{База данных студентов} |
|:---|:---:|:---:|
| Имя | Возраст | Город |
| Иван | 25 | Москва |
| Мария | 30 | СПб |

Параметры:
$$n = 3,\ \text{align} = c,\ \text{text} = \text{"База данных студентов"}$$

### 5.3 Изменение выравнивания отдельной ячейки

Команда \multicolumn также может использоваться для изменения выравнивания одной ячейки:

| Имя | Возраст | Город |
|:---:|:------:|:-----:|
| Иван | 25 | Москва |
| Мария | 30 | СПб |

## 6. Таблицы с фиксированной шириной колонок

Для создания колонок с фиксированной шириной используется тип $p\{w\}$:

$$ \text{Тип колонки} = p\{w\} $$

где $w$ — ширина колонки.

Пример таблицы с колонками фиксированной ширины:

| Термин | Определение |
|:-------|:------------|
| LaTeX | Система компьютерной верстки, которая широко используется для создания научных документов |
| TeX | Язык программирования и система верстки, разработанная Дональдом Кнутом |

Параметры ширины:
$$w_1 = 3\text{ см},\quad w_2 = 4\text{ см}$$

## 7. Добавление пространства между строками

Для улучшения читаемости таблицы можно добавить дополнительное пространство между строками с помощью команды \addlinespace:

| Животное | Описание |
|:---------|:---------|
| Собака | Домашнее животное, друг человека |
| | |
| Кошка | Домашнее животное, ловит мышей |
| | |
| Лошадь | Крупное травоядное животное |

## 8. Числовое выравнивание в колонках

Для правильного выравнивания чисел можно использовать пакет siunitx с типом колонки $S$:

| Элемент | Значение |
|:--------|---------:|
| A | 123.45 |
| B | 6.789 |
| C | 0.1234 |

# Выводы

В ходе выполнения лабораторной работы были изучены основные возможности создания и форматирования таблиц в LaTeX:

**Типы выравнивания колонок:**

$$ \text{Выравнивание} \in \{l,\ c,\ r\} $$

**Профессиональные линии:**

$$ \text{Линии} = \{\text{\toprule}, \text{\midrule}, \text{\bottomrule}\} $$

**Объединение ячеек:**

$$ \text{\multicolumn}\{n\}\{\text{align}\}\{\text{text}\} $$

**Фиксированная ширина:**

$$ \text{Тип колонки} = p\{w\} $$

**Поведение при некорректном количестве элементов:**

- При недостатке элементов: смещение данных
- При избытке элементов: лишние элементы вне таблицы

**Наблюдения:**
- Выбор типа выравнивания существенно влияет на визуальное восприятие таблицы
- Для текстовых данных лучше использовать выравнивание влево
- Для числовых данных — выравнивание вправо
- Для заголовков — выравнивание по центру
- Пакет booktabs обеспечивает типографское качество линий

Полученные навыки позволяют создавать профессионально оформленные таблицы для научных и учебных документов.
