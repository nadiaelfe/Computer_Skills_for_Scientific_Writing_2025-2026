# Выполнение лабораторной работы

## 1. Типы выравнивания колонок

В LaTeX существуют три основных типа выравнивания колонок:

$$ \text{Типы колонок} = \{l,\ c,\ r\} $$

где:
- $l$ — выравнивание влево (left)
- $c$ — выравнивание по центру (center)
- $r$ — выравнивание вправо (right)

### 1.1 Выравнивание влево (l)

При использовании левого выравнивания текст в колонках прижимается к левому краю:

| Имя | Фамилия | Возраст |
|:----|:--------|--------:|
| Иван | Петров | 25 |
| Мария | Иванова | 30 |
| Петр | Сидоров | 28 |

### 1.2 Выравнивание по центру (c)

При использовании центрирования текст располагается по центру каждой колонки:

| Имя | Фамилия | Возраст |
|:---:|:-------:|:-------:|
| Иван | Петров | 25 |
| Мария | Иванова | 30 |
| Петр | Сидоров | 28 |

### 1.3 Выравнивание вправо (r)

При использовании правого выравнивания текст прижимается к правому краю колонок:

| Имя | Фамилия | Возраст |
|----:|--------:|--------:|
| Иван | Петров | 25 |
| Мария | Иванова | 30 |
| Петр | Сидоров | 28 |

## 2. Профессиональные линии в таблицах

Для создания профессиональных горизонтальных линий используются специальные команды:

$$ \text{Типы линий} = \{\text{верхняя}, \text{средняя}, \text{нижняя}\} $$

Пример таблицы с профессиональными линиями:

| Страна | Столица | Население |
|:-------|:--------|----------:|
| Россия | Москва | 146 |
| Франция | Париж | 67 |
| Япония | Токио | 125 |

## 3. Эксперименты с количеством элементов

### 3.1 Недостаточное количество элементов

При создании таблицы с тремя колонками, если в строке указано меньше элементов:

$$ \text{Колонок определено} = 3 $$

$$ \text{Элементов в строке} = 2 $$

Пример некорректной таблицы:

| Имя | Возраст | Город |
|:----|:-------:|:------|
| Иван | 25 | |
| Мария | 30 | Москва |

**Результат:** Возникает ошибка, данные смещаются в неправильные колонки.

### 3.2 Избыточное количество элементов

При указании большего количества элементов, чем определено колонок:

$$ \text{Колонок определено} = 3 $$

$$ \text{Элементов в строке} = 5 $$

$$ \text{Дополнительные элементы} = 2 $$

Пример некорректной таблицы:

| Имя | Возраст | Город |
|:----|:-------:|:------|
| Иван | 25 | Москва | доп1 | доп2 |
| Мария | 30 | СПб | |

**Результат:** Лишние элементы выводятся за пределами таблицы.

## 4. Объединение ячеек

Для объединения нескольких колонок в одну используется специальная команда:

$$ \text{Объединение}(n, \text{align}, \text{text}) $$

где:
- $n$ — количество объединяемых колонок
- $\text{align}$ — выравнивание
- $\text{text}$ — отображаемый текст

### 4.1 Объединение двух колонок

| Личная информация | Город |
|:---:|:---:|
| Имя | Возраст | |
| Иван | 25 | Москва |
| Мария | 30 | СПб |

### 4.2 Объединение всех трех колонок

| База данных студентов |
|:---:|
| Имя | Возраст | Город |
| Иван | 25 | Москва |
| Мария | 30 | СПб |

## 5. Таблицы с фиксированной шириной колонок

Для создания колонок с фиксированной шириной используется специальный тип:

$$ \text{Тип колонки} = p(w) $$

где $w$ — ширина колонки.

Пример таблицы с колонками фиксированной ширины:

| Термин | Определение |
|:-------|:------------|
| LaTeX | Система компьютерной верстки для научных документов |
| TeX | Язык программирования и система верстки |

Параметры ширины:
$$w_1 = 3\text{ см},\quad w_2 = 4\text{ см}$$

## 6. Добавление пространства между строками

Для улучшения читаемости таблицы можно добавить дополнительное пространство между строками:

| Животное | Описание |
|:---------|:---------|
| Собака | Домашнее животное, друг человека |
| | |
| Кошка | Домашнее животное, ловит мышей |
| | |
| Лошадь | Крупное травоядное животное |
