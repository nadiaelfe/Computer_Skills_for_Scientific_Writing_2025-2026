# Выполнение лабораторной работы

## 1. Масштабирование

Размер изображения можно выразить как:

\[
k = \frac{h_{new}}{h_{original}}
\]

## 2. Пропорции

Сохранение пропорций:

\[
\frac{width}{height} = const
\]

## 3. Размещение объектов

Положение объекта можно описать:

\[
P = (x, y)
\]

## 4. Связь с текстом

Используются ссылки:

\[
ref(i) \rightarrow объект_i
\]

Таблица описывается как:

\[
T = \{a_{ij}\}
\]

где \( i \) — строка, \( j \) — столбец.

---

## 2. Размер таблицы

\[

\]

## 3. Выравнивание
